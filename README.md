Objetivos do projeto

- Documentar a API utilizando Swagger/OpenAPI
- Containerizar a aplicação com Docker
- Utilizar PostgreSQL como banco de dados principal
- Implementar versionamento de banco de dados com Flyway
- Aplicar autenticação e autorização com Spring Security
- Implementar arquitetura Multi-Tenant baseada em Schema
- Estruturar o projeto seguindo padrões utilizados em sistemas de grande porte
- Desenvolver seguindo princípios de Clean Code e boas práticas de arquitetura
